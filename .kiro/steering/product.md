# Product overview

## Purpose
SME Knowledge Agent is a conversational AI system that lets employees at small and medium enterprises query scattered company documents — policy PDFs, pricing Excel files, and email threads — in natural language and receive precise, cited answers. The system detects contradictions between documents, resolves them using configurable governance rules, and surfaces its reasoning transparently.

## Target users
- Standard employees (sales reps, support staff) — query knowledge, create tickets
- Knowledge managers (office managers) — upload and manage documents
- Team leads (sales managers) — audit conflict resolutions, override decisions
- System admins — configure workspace, set governance rules, manage roles

## Key features
1. Multi-format ingestion: PDF sections, Excel rows, email threads
2. Hybrid retrieval: vector search + structured metadata for exact citations
3. Autonomous conflict detection with date-priority resolution
4. Role-based access control per department
5. CRM ticket auto-population from query context
6. Google Workspace integration (Drive + Gmail)
7. Admin conflict governance configuration

## Success metrics
- Query response in under 3 seconds
- Citation precision: exact PDF section / Excel row number / email subject
- 100% of contradictions surfaced, never silent
- Employee "Accept AI answer" rate above 90%
