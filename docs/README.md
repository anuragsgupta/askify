- All test files can be run from the project root using `python3 docs/tests/<test_file>.py`
- Shell scripts should be run from the project root: `bash docs/tests/<script>.sh`
- Documentation is organized by type (guides, summaries, tests) for easy navigation
- Keep this folder updated as new features are added
